# EXP-2-PROMPT-ENGINEERING-

## Aim: 
Comparative Analysis of different types of Prompting patterns and explain with Various Test Scenarios

Experiment:
Test and compare how different pattern models respond to various prompts (broad or unstructured) versus basic prompts (clearer and more refined) across multiple scenarios. 
Analyze the quality, accuracy, and depth of the generated responses.


## Algorithm:

Step 1: Define the Topic.Explain the concept of Artificial Intelligence.

Step 2: Use a Broad / Unstructured Prompt.Tell me about AI.
Step 3: Use a Refined / Structured Prompt. Explain Artificial Intelligence with definition, types, applications, advantages, disadvantages, and real-world examples in 250 words.

Step 4: Zero-Shot Prompting Test.What is Machine Learning? Explain briefly.
Step 5: Few-Shot Prompting Test

Example 1: Computer → Electronic device for processing data.
Example 2: Internet → Global network connecting computers.
Now explain “Cloud Computing” in a similar format.

 Step 6: Chain-of-Thought Prompting Test
 A train travels 60 km in 1 hour. How far will it travel in 5 hours? Solve step by step.

 Step 7: Instruction-Based Prompting Test
Write a Python program to check whether a number is prime or not. Include comments and sample output.

Step 8: Creative Scenario Testing
Write a short story about a world without electricity in around 300 words with a moral lesson.

Step 9: Response Comparison Prompt
Compare the outputs generated from broad prompts and refined prompts based on clarity, relevance, accuracy, creativity, and depth of explanation.

 Step 10: Final Analysis Prompt
 
Analyze which prompting pattern produces the most effective response across different scenarios and explain why refined prompts improve LLM performance.

## Result

The experiment showed that refined and structured prompts generated more accurate, detailed, and relevant responses than broad or unstructured prompts. Different prompting patterns improved the clarity, depth, and overall quality of LLM outputs across various scenarios.
